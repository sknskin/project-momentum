# 기획서

## Momentum Scanner — 프로젝트 기획서

### 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Momentum Scanner |
| 유형 | 미국 주식 프리마켓 모멘텀 스크리너 웹 앱 |
| 목적 | 프리마켓 시간대 강세 종목을 빠르게 분석하여 데이 트레이딩 진입 판단 보조 |
| 대상 사용자 | 미국 주식 프리마켓 트레이더, 모멘텀/갭 트레이더 |
| 핵심 가치 | 7가지 정량적 기준으로 종목 모멘텀을 점수화하여 객관적 판단 근거 제공 |

### 2. 배경 및 필요성

프리마켓(장전, 04:00~09:30 ET) 시간대는 다음과 같은 특성이 있습니다:

- **뉴스, 실적 발표 등의 촉매(Catalyst)**로 큰 가격 변동 발생
- **거래량이 적어** 변동성이 크고, 가격 갭(Gap)이 자주 발생
- **빠른 판단**이 필요하지만, 수동으로 여러 종목을 분석하기 어려움

이 스크리너는 프리마켓 데이터를 자동 수집·분석하여, 트레이더가 빠르게 유망 종목을 필터링할 수 있도록 합니다.

### 3. 핵심 기능

#### 3.1 종목 스캔

- 사용자가 직접 종목 심볼 입력 (콤마 구분)
- 인기 종목(Hot Tickers) 프리셋 원클릭 스캔
- 60초 자동 갱신으로 실시간 모니터링

#### 3.2 7가지 모멘텀 스코어링

각 기준은 0/1/2점 체계로, 명확하고 직관적입니다.

| # | 기준명 | 0점 | 1점 | 2점 |
|---|--------|-----|-----|-----|
| C1 | Pre-Market Change | 변동 <1% | 1~3% | >3% |
| C2 | Volume Surge | 비율 <1x | 1~2x | >2x |
| C3 | Gap Size | 갭 <2% | 2~5% | >5% |
| C4 | 52W High Proximity | >10% 거리 | 5~10% | <5% |
| C5 | Catalyst Events | 없음 | 1~2개 | 3개+ |
| C6 | Social Buzz | 낮음 (<40) | 보통 (40~69) | 높음 (70+) |
| C7 | Price Stability | 변동 >5% | 2~5% | <2% |

> **C6 Social Buzz 참고:** 현재 시뮬레이션 데이터를 사용하며, 실제 소셜 미디어 분석이 아닙니다.

#### 3.3 시그널 판정

```
HIGH (10~14점)     → 🔥 강한 모멘텀, 진입 적극 고려
MODERATE (6~9점)   → ⚡ 보통 모멘텀, 확인 후 진입
LOW (0~5점)        → 💤 약한 모멘텀, 관망 권장
```

#### 3.4 자동 엔트리 전략 생성

시그널에 따라 자동으로 진입 전략을 생성합니다:

| 항목 | HIGH | MODERATE | LOW |
|------|------|----------|-----|
| Stop Loss | -2% | -3% | -5% |
| Target 1 | +3% | +2% | +1% |
| Target 2 | +6% | +4% | +2% |

#### 3.5 마켓 세션 감지

미국 동부시간(ET) 기준으로 현재 세션을 자동 감지합니다:

- **PRE-MARKET** (04:00~09:29): 시안 배지 + 펄스 애니메이션
- **MARKET OPEN** (09:30~15:59): 녹색 배지
- **AFTER HOURS** (16:00~19:59): 주황 배지
- **CLOSED** (20:00~03:59): 회색 배지

### 4. 데이터 소스

| 데이터 | 소스 | 방식 |
|--------|------|------|
| 주가, 거래량, 52주 고저 | Yahoo Finance API v8 | 서버 프록시 |
| 소셜 버즈 | 시뮬레이션 | 심볼 해시 기반 생성 |
| 촉매 이벤트 | 자동 감지 | 가격/거래량 이상 기반 |

> Yahoo Finance API 실패 시 자동으로 모의(Mock) 데이터로 전환됩니다.

### 5. UI/UX 설계

#### 5.1 디자인 컨셉

- **다크 테마 전용**: #0f0f0f 배경, 눈의 피로 최소화
- **트레이딩 대시보드 느낌**: 데이터 밀도 높은 카드 레이아웃
- **색상 코딩 일관성**: 상승=초록, 하락=빨강, 시안=액센트

#### 5.2 색상 팔레트

| 용도 | 색상 | Hex |
|------|------|-----|
| 배경 | 다크 | #0f0f0f |
| 카드 배경 | 네이비 다크 | #1a1a2e |
| 강한 모멘텀 | 레드 | #ff4d4d |
| 보통 모멘텀 | 앰버 | #f5a623 |
| 약한 모멘텀 | 그레이 | #666666 |
| 본문 텍스트 | 라이트 그레이 | #e0e0e0 |
| 액센트 | 시안 | #00d4ff |

#### 5.3 반응형 설계

- **데스크톱** (md 이상): 종목 카드 2열 그리드
- **모바일** (md 미만): 종목 카드 1열 세로 배치
