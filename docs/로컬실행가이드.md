# 로컬 실행 가이드

## Momentum Scanner — 로컬 개발 환경 설정

### 1. 사전 요구사항

| 항목 | 최소 버전 | 확인 명령 |
|------|-----------|-----------|
| Node.js | 18.x | `node -v` |
| pnpm | 9.x | `pnpm -v` |
| Git | 2.x | `git --version` |

### 2. 프로젝트 클론 및 설치

```bash
# 프로젝트 디렉토리로 이동
cd /path/to/projects

# 저장소 클론
git clone <repository-url> momentum
cd momentum

# 의존성 설치
pnpm install
```

### 3. 개발 서버 실행

```bash
# 개발 모드 실행 (Hot Reload 지원)
pnpm dev
```

실행 후 브라우저에서 `http://localhost:3000` 에 접속합니다.

### 4. 프로덕션 빌드 및 실행

```bash
# 프로덕션 빌드
pnpm build

# 프로덕션 서버 시작
pnpm start
```

### 5. 환경 변수

현재 이 프로젝트는 별도의 환경 변수 설정이 필요하지 않습니다.
Yahoo Finance API는 공개 API를 사용하며, 별도의 API 키가 필요하지 않습니다.

> **참고:** Yahoo Finance API가 차단되거나 응답하지 않을 경우, 자동으로 모의(Mock) 데이터로 전환됩니다. 이 경우 종목 카드에 `[DEMO MODE]` 배지가 표시됩니다.

### 6. 사용 방법

1. 상단 입력 필드에 종목 심볼을 쉼표로 구분하여 입력합니다.
   - 예: `TSLA, NVDA, AAPL`
2. `Scan` 버튼을 클릭하거나 Enter 키를 누릅니다.
3. 또는 `Hot Tickers` 버튼을 클릭하면 인기 종목이 자동으로 스캔됩니다.
4. 60초마다 자동으로 데이터가 갱신됩니다.
5. 카운트다운 버튼을 클릭하면 즉시 새로고침됩니다.

### 7. 프리마켓 시간대

이 스크리너는 미국 동부시간(ET) 기준으로 동작합니다:

| 세션 | 시간 (ET) | 배지 색상 |
|------|-----------|-----------|
| PRE-MARKET | 04:00 ~ 09:29 | 시안 (펄스 애니메이션) |
| MARKET OPEN | 09:30 ~ 15:59 | 녹색 |
| AFTER HOURS | 16:00 ~ 19:59 | 주황 |
| CLOSED | 20:00 ~ 03:59 | 회색 |

### 8. 문제 해결

#### 빌드 오류 발생 시

```bash
# node_modules 삭제 후 재설치
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

#### 포트 충돌 시

```bash
# 다른 포트로 실행
pnpm dev -- -p 3001
```

#### Yahoo Finance API 응답 없음

Yahoo Finance API는 지역 또는 네트워크에 따라 차단될 수 있습니다.
이 경우 모의 데이터가 자동으로 사용되며, 종목 카드에 `[DEMO MODE]`가 표시됩니다.
모의 데이터는 현실적인 범위 내에서 랜덤 생성되며, 실제 시세와 무관합니다.

### 9. 린트 검사

```bash
pnpm lint
```
